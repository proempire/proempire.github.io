react基础回顾~~

一、配置环境和依赖

react使用jsx语法，故使用type = ‘ babel ‘ 注明，使用react.js, react-dom.js, browser.js三个库。

二、基础使用

  1. 基础方法： ReactDOM.render()

    ReactDOM.render(
  <h1>Hello, world!</h1>,
  document.getElementById('example')
);

  2. jsx语法示例：

    <div>{ oarray.map(function(el)) }</div>

  3. 封装component方法——React.createClass():

    var realwhat = function(){return <h3>{this.props.name}</h3>}
var owhat = {render: realwhat}
var shakeIt = React.createClass(owhat)

ReactDOM.render({<shakeIt name='emperor'>, document.get...})
    • 组件类首字母必须大写，上面这个例子是错的

  4. this.props.children获取所有子节点，类型由节点数量决定，PropTypes用于验证属性参数类型是否合法，用法：

    react.createClass({
    render: function(){
        return react.propTypes.【类型】.isrequired
    }
});

    react模板语言中引用变量时用{}包裹

  5. react使用dom-diff，可极大提高网页性能，若想获取真实DOM节点，使用this.ref.[refname]，注意，ref属性应给予dom节点。

    • 虚拟DOM未插入文档时不能获取真实dom节点

  6. react支持状态机，getInitialState和setState用于处理状态，注意它与props的区别

  7. 与表单内容的互动，关键部分如下，备查:

    handleChange: function(event) {
  this.setState({value: event.target.value});
},
render: function () {
  var value = this.state.value;
  return (
    <div>
      <input type="text" value={value} onChange={this.handleChange} />
      <p>{value}</p>
    </div>
  );
}

  8. 组件的生命周期中，有两个特殊状态处理：收到新参数，是否重新渲染
