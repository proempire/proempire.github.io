vue-day1

一、vue.js概述

这是一个MVVM框架的实例，vue扮演者ViewModel的角色，DOM监听数据变化，数据绑定到DOM上。

二、vue实例创建

var MyComponent = Vue.extend({
  // 扩展选项
})

// 所有的 `MyComponent` 实例都将以预定义的扩展选项被创建
var myComponentInstance = new MyComponent()

只有在创建时传入的数据才是响应式的，后面添加的属性不会触发视图更新。

vue暴露了一些属性和方法，均加以$前缀，如$watch方法：

// $watch 是一个实例方法
vm.$watch('a', function (newVal, oldVal) {
  // 这个回调将在 `vm.a`  改变后调用
})

三、数据绑定语法

  1. vue的template基于DOM而不是字符串。
  2. Mustache标签：
  3. 只处理单次插值，在Mustache前加上*即可。
  4. Mustache里的内容称为绑定表达式。
  5. 指令的参数是特性转化后的形式，插值可以应用于特性内。
  6. v-bind和v-on均可以缩写，前缀分别为:和@。

四、计算属性

var vm = new Vue({
  el: '#example',
  data: {
    a: 1
  },
  computed: {
    // 一个计算属性的 getter
    b: function () {
      // `this` 指向 vm 实例
      return this.a + 1
    }
  }
})
